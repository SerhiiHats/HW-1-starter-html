Привет ITVDN

С развитием приложения Telegram и расширением круга его аудитории, массовым и распространённым стало использование такого нужного и полезного функционала как Телеграмм боты.

С помощью Телеграмм ботов можно доставлять пользователю нужную ему информацию, вести записки, блокноты, вести учет новых выученных слов по английскому языку, записывать их, заказывать пиццу, такси, любую другую услугу, можно передавать любые данные, например, собирать коммунальным предприятиям от пользователя данные их приборов учета (электроэнергия, вода, тепло) без посещения пользователями этих организаций, а также развивая это направление можно напоминать пользователям об необходимости передать показания приборов учета, выставлять счета на оплату, делать рассылку рекламы или напоминать о задолженности, и т.д.

Возможности Телеграмм ботов обширны и широко используются в продакшене.

Рассмотрим процесс создания, для начала, простого Телеграмм бота с помощью Java и Фреймворка Spring Boot.
